# verificador-de-nomes-python
Analise de dois arrays contendo nomes com caracteres especiais

Código desenvolvido para solucionar um problema de análise de vários nomes dentro do excel.
A lista oficial foi passada e foi armazenada em um array (excelDados), já os nomes para comparação foram retirados usando uma busca através do HTML da página (talvez ter criado um robo para pegar esses dados seria melhor).

Para garantir a certeza entre as duas listas foi feito o tratamento em python para que os caracteres especiais fossem devidamente tratados. O resultado foi a comparação entre os dois arrays os nomes presentes nas duas listas e os não presentes. Evitando assim, a análise de manual de um por um.
